# 얼굴 인식 체크 일시 중단 및 복구 가이드

본 문서는 실전 면접 진행 시 얼굴 인식(영상 테스트) 실패로 인해 다음 단계로 넘어가지 못하는 현상을 해결하기 위한 임시 조치 및 복구 방법을 설명합니다.

---

### 1. 수정 목표

- 환경 테스트 마지막 단계(`FinalGuidePage`)에서 영상 인식 결과가 실패(`false`)하더라도 [지금 면접 시작하기] 버튼이 활성화되도록 수정.

### 2. 대상 파일

- `frontend/src/pages/interview/frontend/src/pages/landing/FinalGuidePage.jsx`

### 3. 수정 예정 내용 (Plan)

#### [기존 코드]

```javascript
const isAudioOk = sessionStorage.getItem('env_audio_ok') === 'true';
const isVideoOk = sessionStorage.getItem('env_video_ok') === 'true';
const allPassed = isAudioOk && isVideoOk;
```

#### [수정 코드 - 임시]

```javascript
const isAudioOk = sessionStorage.getItem('env_audio_ok') === 'true';
// const isVideoOk = sessionStorage.getItem('env_video_ok') === 'true';
const isVideoOk = true; // [임시] 얼굴 인식 체크 무시
const allPassed = isAudioOk && isVideoOk;
```

### 4. 복구 방법 (나중에 다시 켤 때)

기능을 다시 활성화하려면 다음 단계를 따르세요.

1. `frontend/src/pages/landing/FinalGuidePage.jsx` 파일을 엽니다.
2. `// [임시] 얼굴 인식 체크 무시` 라고 적힌 줄을 삭제합니다.
3. 바로 윗줄의 주석(`//`) 처리된 원래 코드(`const isVideoOk = ...`)의 주석을 제거하여 활성화합니다.
4. 파일을 저장하면 즉시 얼굴 인식 체크가 필수 항목으로 복구됩니다.

---

**주의**: 이 조치는 테스트 편의를 위한 임시 방편입니다. 실제 서비스 배포 시에는 반드시 복구해야 합니다.
