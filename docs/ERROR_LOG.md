## 🛠️ 에러 발생 시 기록 가이드
에러가 발생하면 위 리스트에 추가하고, 상세 내용이 필요할 경우 아래에 섹션을 추가하여 기록해 주세요.

### 상세 기록 양식 (선택 사항)
1. **에러명**: 발생한 에러의 이름 (예: `ValueError`, `FileNotFoundError`)
2. **발생원인**: 에러가 발생한 구체적인 이유
3. **조치내역**: 에러를 해결하기 위해 수행한 작업
4. **발생위치**: 코드 내 위치 (클래스, 함수, 라인 번호 등)
5. **실행파일명**: 에러가 발생한 파일의 경로 및 파일명
6. **실행시간**: 에러가 처음 발생한 시간
7. **수정시간**: 에러를 해결하고 수정을 마친 시간
8. **최종정상실행시간**: 해당 모듈이 마지막으로 성공적으로 실행되었던 시간
