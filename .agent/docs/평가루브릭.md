A. 자기 표현 & 기본 커뮤니케이션 (15%)
구분	내용
평가 영역	자기 표현 능력, 기본 커뮤니케이션
대상 단계	자기 소개 단계(선택) 최종 자유 발언 단계
평가 목적	지원자의 배경과 강점을 빠르고 명확하게 전달하는 능력 평가
평가 기준	• 본인 배경과 강점이 명확히 전달되는가
• 답변 구조가 이해하기 쉬운가
• 질문 의도에서 벗어나지 않는가
LLM 관찰 포인트	• 서론–본론–결론 구조 존재 여부
• 핵심 키워드 반복 및 강조 여부
감점 요인	• 장황하지만 핵심이 없음
• 질문과 무관한 이야기 반복

B. 지원 동기 & 회사 적합성 (15%)
구분	내용
평가 영역	지원 동기, 회사·직무 적합성
대상 단계	지원 동기 단계
평가 목적	회사와 직무를 이해한 상태에서 지원했는지 판단
평가 기준	• 지원 직무를 정확히 이해하고 있는가
• 회사/도메인 언급이 구체적인가
• 단순 열정이 아닌 명확한 선택 이유가 있는가
LLM 관찰 포인트	• 회사 인재상 키워드와의 정렬 여부
• 이력서 프로젝트·경험과 동기의 연결성
감점 요인	• 어디든 쓸 수 있는 범용 동기
• 회사명만 바꿔도 성립하는 답변

C. 직무 지식 이해도 (20%)
구분	내용
평가 영역	직무 관련 이론·개념 이해
대상 단계	직무 관련 지식 질문직무 관련 지식 추가 질문 (1-1)
평가 목적	직무 수행에 필요한 기본 지식 수준 검증
평가 기준	• 기본 개념을 정확히 설명했는가
• 용어를 맥락에 맞게 사용했는가
• 추가 질문 후 설명이 개선되었는가 
LLM 관찰 포인트	• 개념 오류 여부
• 재질문 후 답변 명확성 변화
감점 요인	• 용어 나열만 하고 설명 불가
• 재질문 후에도 동일한 모호한 답변


D. 직무 경험 & 문제 해결 (30%)
구분	내용
평가 영역	실무 경험, 문제 해결 사고력
대상 단계	직무 관련 경험 질문직무 관련 문제 해결 질문추가 질문 (2-1, 3-1)
평가 목적	실제로 문제를 해결해 본 경험과 사고 흐름 평가
평가 기준	• 실제 경험 기반 설명인가
• 문제 정의 → 접근 → 결과 흐름이 있는가
• 대안·개선 관점이 있는가
• 추가 질문 후 논리 보완이 되었는가 
LLM 관찰 포인트	• “내가 했다” 중심 서술 여부
• 추상적 설명 vs 구체적 행동
감점 요인	• 팀이 했다는 이야기만 반복
• 결과 없는 과정 설명


E. 인성 & 성장 가능성 (20%)
구분	내용
평가 영역	태도, 협업, 성장 가능성
대상 단계	협업 평가 질문 (+1-1)책임감·가치관 질문 (+2-1)변화 수용·성장 질문 (+3-1)
평가 목적	조직 적합성과 장기 성장 가능성 판단
평가 기준	• 협업 시 태도가 성숙한가
• 책임 회피 없이 설명하는가
• 실패·변화를 학습 관점으로 해석하는가
• 추가 질문 후 태도 설명이 명확해졌는가 
LLM 관찰 포인트	• blame language vs ownership language
• “배웠다 / 개선했다” 표현 사용 여부
감점 요인	• 타인·환경 탓 중심 설명
• 실패 경험 회피

LLM 평가 프롬프트
