1. 시스템 개요

본 문서는 pdfplumber 기반 이력서 자동 파싱 시스템의 기술적 구조와 동작 원리를 설명하는 종합 보고서입니다.

본 시스템은 PDF 내부 객체(Object) 기반 텍스트·표 추출 및 추출 데이터를 구조화된 JSON 형태로 변환하는 작업을 수행합니다.

2. PDF 파싱 엔진 핵심 원리

  • 객체 기반 추출: OCR 방식이 아닌 내부 객체 데이터를 직접 분석하여 텍스트와 좌표 정보를 획득합니다.
  • 좌표 시스템 기반 구조 인식: 텍스트 문장 인식 및 선 교차 지점 감지를 통한 표(Table) 인식 기능이 강력합니다.
