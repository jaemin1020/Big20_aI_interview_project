### 📝 프로젝트 커밋 컨벤션 (Commit Convention)

| Type | 설명 | 비고 |
| :--- | :--- | :--- |
| data | 데이터 처리 과정 중 완료된 작업 커밋 | 데이터 수집 및 전처리 완료 시 |
| wip | 진행 중인 작업 커밋 (main 브랜치 금지) | Work In Progress (작업 중) |
| feat | 새로운 기능 추가 커밋 | 기능 구현 완료 시 |
| docs | 문서 수정 및 추가 커밋 | README, 위키 등 문서 작업 |
| test | 전처리 및 모델 실제 테스트 후 결과 커밋 | 모델 검증 및 테스트 결과물 |
| perf | 성능 조정 및 모델 튜닝 작업 커밋 | 하이퍼파라미터 튜닝, 속도 개선 |
| revert | 이전 커밋 되돌림 | 이전 상태로 롤백이 필요할 때 |
| fix | 오류 해결 및 기존 작업 수정 후 커밋 | 버그 수정 및 로직 보완 |
| merge | 개인 브랜치에 merge로 작업 내용을 받을 때 | 브랜치 통합 시 사용 |
| exp | Git 테스트 및 예외 사항 전용 (진행 후 사용 금지) | 실험적 코드, 오류/에러 로그 등 |
