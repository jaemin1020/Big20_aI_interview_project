# STT Task 모듈 (stt.py)

지원자의 음성 답변을 텍스트로 변환하는 비동기 작업을 수행합니다.

## 주요 기능

### 1. `tasks.stt.recognize`
- **설명**: 오디오 데이터를 텍스트로 변환(STT)합니다.
- **사용 기술**: `Faster-Whisper` (large-v3-turbo 모델)
- **특징**:
    - `int8` 양자화 및 CPU 최적화로 처리 속도 향상
    - **환각(Hallucination) 방지**: 무음 구간이나 의미 없는 반복 문구 필터링 로직 포함
    - **메모리/파일 하이브리드 처리**: 우선 메모리(`BytesIO`)에서 처리하고 실패 시 임시 파일로 전환하여 안정성 확보
    - `vad_filter`를 적용하여 노이즈 및 무음 구간 필터링
- **매개변수**:
    - `audio_b64`: Base64 형태의 오디오 데이터
- **결과**: 변환된 텍스트(`text`)를 반환합니다.
