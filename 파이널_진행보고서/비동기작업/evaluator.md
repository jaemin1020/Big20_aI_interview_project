# Evaluator Task 모듈 (evaluator.py)

지원자의 답변을 정밀 분석하고 점수를 산출하며, 면접 종료 후 최종 보고서를 생성합니다.

## 주요 기능

### 1. `tasks.evaluator.analyze_answer`
- **설명**: 개별 질문에 대한 답변을 스테이지별 루브릭(평가 기준)에 따라 점수화(0~100점)하고 피드백을 생성합니다.
- **특징**:
    - 질문 성격(9~14번 스테이지 등)에 따라 회사의 **인재상(Ideal)**을 반영하여 평가
    - 감성 분석 및 루브릭 세부 항목별 점수 산출
    - 결과는 `Transcript`에 저장되어 결과 화면에서 가중치 합산에 활용됨
- **큐(Queue)**: `gpu_queue`

### 2. `tasks.evaluator.generate_final_report`
- **설명**: 면접이 종료된 후 전체 대화 내용을 종합 분석하여 '시니어 면접 위원장' 페르소나의 최종 리포트를 생성합니다.
- **분석 영역**: 기술 역량, 프로젝트 경험, 문제 해결 능력, 의사소통, 조직 적합성 등
- **출력**: 강점/보완점(근거 중심), 종합 점수, 시니어 제언이 담긴 JSON 데이터를 생성하여 DB에 저장
- **큐(Queue)**: `gpu_queue`
