# TTS Task 모듈 (tts.py)

AI 면접관의 질문 텍스트를 음성(Audio) 데이터로 변환하는 비동기 작업을 수행합니다.

## 주요 기능

### 1. `tasks.tts.synthesize`
- **설명**: 텍스트를 고품질 음성으로 합성하여 WAV 파일로 저장하고 결과를 반환합니다.
- **사용 기술**: `Supertonic 2` 엔진
- **특징**:
    - 싱글톤 패턴으로 엔진 관리
    - `tts_lock`을 통한 스레드 안전성 보장 (CPU 경합 방지)
    - 중복 생성 방지를 위한 파일 존재 확인 로직 포함
    - 생성된 파일은 `/app/uploads/tts/` 경로에 저장되어 백엔드에서 서빙됨
- **매개변수**:
    - `text`: 음성으로 변환할 질문 내용
    - `language`: 언어 설정 (기본값: "ko")
    - `question_id`: DB 저장 및 파일 매칭을 위한 질문 ID
- **큐(Queue)**: `cpu_queue`
